API Information

POST https://oa-madison-2024.onrender.com/apply

To submit a new application, send a POST request to the https://oa-madison-2024.onrender.com/apply endpoint with the following JSON structure: 

{
  "first_name": "String",
  "last_name": "String",
  "email": "String",
  "phone_number": "String",
  "year": Number, // Expected graduation year
  "why_do_you_want_to_join": "String",
  "urls": ["String"] // Include "https://"
}

The API will return a JSON response with the following structure: 

{
  "message": "Application submitted successfully",
  "applicationId": "YOUR_APPLICATION_ID"
}

PATCH https://oa-madison-2024.onrender.com/apply

To update an existing application, send a PATCH request to the https://oa-madison-2024.onrender.com/apply endpoint with the following JSON structure: 

{
  "applicationId": "String",
  "applicationData": {
    "first_name": "String",
    "last_name": "String",
    "email": "String",
    "phone_number": "String",
    "year": Number, // Expected graduation year
    "why_do_you_want_to_join": "String",
    "urls": ["String"]
  }
}

The API will return a JSON response with the following structure: 

{
  "message": "Application updated successfully",
  "applicationId": "YOUR_APPLICATION_ID"
}

DELETE https://oa-madison-2024.onrender.com/apply

To delete an existing application, send a DELETE request to the https://oa-madison-2024.onrender.com/apply endpoint with the following JSON structure: 

{
  "applicationId": "String"
}

The API will return a JSON response with the following structure: 

{
  "message": "Application deleted successfully"
}

Error Codes and Troubleshooting

If you encounter an error while using our API, please refer to the following error codes and instructions:

INVALID_REQUEST (400)

This error occurs when the request is missing required fields or contains invalid data.

What to do: Check the error message for specific details about which field is missing or incorrectly formatted. Ensure all required fields are present and correctly formatted according to the API Information section above.

DATABASE_ERROR (500)

This error indicates a problem with our database while processing your application.

What to do: Please try submitting your application again after a few minutes. If the problem persists, contact our support team.

RATE_LIMIT_EXCEEDED

You've sent too many requests in a short period of time.

What to do: Wait for at least 15 minutes before trying to submit your application again. If you need to submit multiple applications, please space out your requests.
